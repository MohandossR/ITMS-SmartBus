Intelligent Transportation Management System for Smart Buses.

## Overview
ITMS-SmartBus is a smart bus monitoring system designed to improve passenger safety and transportation efficiency using AI and real-time dashboards.

## Key Features
- Driver drowsiness detection
- Passenger counting
- Live monitoring dashboard
- Safety alerts and notifications

## Tech Stack
- Python
- OpenCV
- Flask
- HTML / CSS
