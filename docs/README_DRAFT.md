# Intelligent Transportation Management System (ITMS)

## Problem
Public buses lack real-time safety and passenger monitoring.

## Solution
A smart bus system that monitors drivers, passengers, and vehicle data using AI and dashboards.

## Features
- Driver drowsiness detection
- Passenger counting
- Live dashboard
- Safety alerts

## Tech Stack
- Python
- OpenCV
- Flask
- HTML/CSS
